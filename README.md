# Smart City Traffic Dashboard

A software-only smart city dashboard that analyzes traffic and road data to provide insights on congestion hotspots, road conditions, and optimized routes for better urban mobility.

## Features
- Traffic density visualization on maps
- Congestion hotspot detection
- Basic route optimization
- City-level traffic insights and analytics

## Tech Stack
- Frontend: React, Leaflet, Chart.js
- Backend: Python, FastAPI
- Database: PostgreSQL / Supabase
- Data Processing: Pandas, NumPy

## Project Structure
- `/frontend` – React frontend
- `/backend` – FastAPI backend
- `/data` – Sample datasets
- `/docs` – Documentation and screenshots
